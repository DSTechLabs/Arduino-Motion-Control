About the DSArm One
The DSArm One is the first six DOF (degrees of freedom)
robotic arm developed by D+S Tech Labs, Inc.

Our goal is to provide a simple, low cost robotic arm
that is still accurate and capable for medium duty
pick-and-place or assembly applications.

Each joint, plus gripper, uses an open-ended micro-stepping motor
controlled by a dedicated Arduino NANO MCU board
running our proprietary Stepper Motor Controller firmware.

About this Motion Controller
The DSArm Motion Controller provides a high level
user interface to the robot arm for manual, scripted and
autonomous motion.

It is an AngularJS Web Application that issues motor
commands to the Stepper Motor Controllers using
high-speed web-socket communication, the Bot Server.
