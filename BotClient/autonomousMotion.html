Autonomous Motion
This section will allow the DSArm to operate in an autonomous mode.

─────────────── Artificial Intelligence ───────────────

Components of this mode will include:
  • A vision system using its EE-Box camera
  • Object recognition and database
  • Haptic feedback from touch/pressure sensors
  • Other sensors may include:
    • laser distance
    • pressure
    • temperature
    • proximity
    • tilt/gyro
    • accelerometer
  • A "Task Definition" area for specifying high-level tasks
    For example: "Move object A to location B"
  • A "Rule Editor" with decision making abilities
    For example: "If Sensor A > 50 then run Script B"
